# 🛍️ Customer Shopping Behavior Analysis

An end-to-end data analytics project that analyzes customer shopping behavior using **Python**, **Microsoft SQL Server**, and **Power BI**. The project focuses on understanding purchasing patterns, customer segmentation, product performance, and subscription behavior to generate actionable business insights.

---

## 📌 Project Overview

Businesses generate massive amounts of transactional data every day. This project transforms raw customer purchase data into meaningful insights through data cleaning, SQL analysis, and interactive dashboards.

The analysis answers business questions such as:

- Which customer groups generate the most revenue?
- Which products receive the highest ratings?
- Do subscribers spend more than non-subscribers?
- Which products depend heavily on discounts?
- Which age groups contribute the most revenue?
- How can customer retention be improved?

## 💡 Key Insights

- Male customers generated higher total revenue.
- Average spending was similar between subscribers and non-subscribers.
- Young customers contributed the highest revenue.
- Some products relied heavily on discounts to drive sales.
- Loyal customers formed the largest customer segment.
- Clothing was the highest-selling product category.
